# Contributing to Toolmetry AI Bot

## Quick Start

1. Fork the repository
2. Clone your fork
3. Create a new branch
4. Make your changes
5. Submit a pull request

## Adding Training Data

Edit `bot/data/public-traindata.json` to add new patterns and responses.

## Example

```json
{
  "knowledgeBase": {
    "myCategory": {
      "patterns": ["keyword1", "keyword2"],
      "responses": {
        "en": "English response",
        "hi": "Hindi response"
      }
    }
  }
}
```

Happy contributing! 🎉
